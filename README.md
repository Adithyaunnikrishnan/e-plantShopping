# e-PlantShopping

## Project Description
e-PlantShopping is a React-based e-commerce web application designed for purchasing indoor and outdoor plants.  
The application allows users to browse different plant categories, add plants to a shopping cart, manage quantities, and view the total cost dynamically.

## Features
- Display multiple plant categories with product cards
- Add plants to the shopping cart
- Disable "Add to Cart" button once an item is added
- Increase or decrease product quantity in the cart
- Remove items from the cart
- Dynamic cart total and item count
- Redux Toolkit used for global state management

## Technologies Used
- React
- Redux Toolkit
- JavaScript (ES6)
- CSS
- GitHub Pages

## Project Purpose
This project was developed as part of an academic assignment to demonstrate:
- Component-based UI development
- State management using Redux
- Shopping cart functionality in a React application
